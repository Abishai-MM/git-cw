Explicación de código en C

Código


    #include <stdio.h>>
    int main()
    {
        int edad;
        printf("¿Cuántos años tienes");
        scanf("%d",&edad);

        if(edad >=18)
        {
            printf("Eres mayor de edad.\n");
        }
        else
        {
            printf("Eres menor de edad.\n");
        }

        return 0;
    }
Explicación del código

La línea #include <stdio.h>> incluye la librería estándar de entrada y salida.

La función int main() es la función principal del programa.

La línea int edad; crea una variable llamada edad de tipo entero, que servirá para guardar la edad del usuario.

La función printf("¿Cuántos años tienes");muestra un mensaje en pantalla preguntando la edad.

La función scanf("%d",edad);sirve para leer un número que escriba el usuario y guardarlo en la variable edad.

La estructura condicional if(edad >=18) comprueba si la edad es mayor o igual a 18.

La función printf("Eres mayor de edad.\n"); muestra el mensaje como resultado de la aprobación de la condición anterior.

La línea else se ejecuta si la condición if es falsa.

La función printf("Eres menor de edad.\n"); muestra el mensaje tras el cumplimiento del else.

La línea return 0; sirve para indicar que el programa terminó correctamente, return regresa un valor y 0 significa que no hubo erores.
